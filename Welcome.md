# Welcome to Quill

Quill is a **WYSIWYG Markdown editor**: what you see is the document, and what
you save is plain Markdown. Try typing `## ` at the start of a line, or
wrapping a word in `**double asterisks**` — the syntax melts away as you type.

## Things to try

- Select some text and use the floating bubble to format it
- Press `⌘K` on a selection to add a [link](https://daringfireball.net/projects/markdown/)
- Press `⌘/` to peek at the Markdown source underneath

Start a line with `- [ ]` to make a task:

- [x] Build a tasteful editor
- [ ] Write something wonderful

> Markdown is intended to be as easy-to-read and easy-to-write as is feasible.
> — John Gruber

## Code

Fenced code blocks are highlighted automatically:

```js
function greet(name) {
  // Quill highlights ~190 languages
  return `Hello, ${name}!`;
}
```
